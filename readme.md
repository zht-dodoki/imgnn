# IMGNN

The code is implemented based on python 3.8 and requirements.txt. Datasets are publicly available at [KONECT](https://dl.acm.org/doi/10.1145/2487788.2488173) and [SNAP](https://arxiv.org/abs/1606.07550).

```
cd imgnnv2
python ./main.py -train_data cora_ml -infer_data youtube -seed_size 1% -infer_model IC
```
